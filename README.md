# CryptoPiggy — web

The production web app: **Preference → preview → execute** (Next.js). Beginner-facing, non-custodial —
the user sets a risk preference & target, previews the Plan the engine returns, and executes it from
their own wallet. Not built yet.

> The local **contracts demo** (the viem single-page app that drives anvil) used to live here. It has
> moved to the contracts repo, where it belongs: [`contracts/demo/`](https://github.com/CryptoPiggy000/contracts/tree/main/demo).
